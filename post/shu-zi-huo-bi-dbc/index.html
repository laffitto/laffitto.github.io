数字货币DBC

· 2020-08-08 · # 新东西 # 折腾 # 投资 # 思考

一个关于人工智能的数字货币。

最近由于算力的原因，一直在找深度学习的云平台，在多方的寻找查阅下，找到了一个叫1024lab的租用GPU算力的平台，注册时发现平台是基于数字货币DBC进行支付使用的，这让我充满了兴趣。

DBC全称为DeepBrain Chair，中文称为深脑链（听起来怪怪的）。现在具体的交易情况可以在这里或这里查到。之前虽然研究过一些数字货币，但都没有深入，自己也注册过imtoken数字钱包，却基本上没有用过，这次由于涉及到一些小项目，需要算力充值，也算是重新倒弄了一番。

数字货币构建于区中心化的网络，依托于区块链，可以得到很好监督作用，最具代表性且被熟知的是比特币（BTC），在我需要对DBC进行充值的时候，根据1024lab的帮助文档，发现有很多种充值方式，但其精髓都是先用实体货币兑换成比较通用的几种数字货币，例如：比特币（BTC），以太坊（ETH），泰达币（USDT）等，而其中BTC是最为通用的（不愧是老大）。实际上这些平台就提供了一个实体兑换数字货币的担保，尽可能减小跑路的风险。

我在浏览之后，选择了火币网这个平台（也是因为很早之前就听说过）。在一系列的注册后，映入眼帘的是类似于股票交易的界面，在探索了各个板块后，真的基本上和股票的交易系统有很高的相似性（或者说港股/美股实际更准确）。各种数字货币，对应的即是各种股票，而像比特币，以太坊这种通用型，更像是美元，港币，因此你在兑换它们的时候是有一些汇率差的。

抱着好奇的心态，我用支付宝买了100元的比特币，实际支付方式之多，速度之快，操作之便捷让我大开眼界。

买完后在我的比特币钱包中便有了对应的数额，这时便可以用比特币进行DBC的交易。这个界面，委托，条件单真的和股市没太大区别🤣，选定交易数量啥之后可以成交。不过也有一些区别，我从网上找了几条（此处仅对应火币网）：

  1. 交易时间：7*24小时全年无休市。
  2. T+0：股票是T+1交易，即当天买入股票，下一个交易日才可卖出。而比特币是T+0交易，当天买入当天即可卖出。
  3. 无涨跌停： 比特币交易无涨跌停限制，股票有涨跌停限制，例如：5月28日比特币单日涨幅超20%。
  4. 交易单位：最小可买0.0001BTC（约0.6元），没有股票最少买一手（100股）的买入限制。
  5. 提币无时间限制：随时提币提现，资金流动性高。
    看完后觉得666，就是一个强化版的股票交易市场，很好奇服务器维护啥的在啥时候进行。😅

当然，我也可以把买到的比特币转到更其他的数字钱包（例如imtoken），在“资产”界面，点击“提币”，输入对应的钱包地址，转出即可。DBC转到1024lab平台也是相同的操作，只要知道钱包的地址就可以了，真的很方便。不过我觉得之前没有接触过这些概念的人难免是会一头雾水的😵。

在我个人看来，现在数字货币并没有被广泛的使用，大家对于数字货币仅仅是知道其名字而不太了解的情况下，究其原因还是其依附于区块链，去中心化。没有任何国家会赞同自己的货币可以随意流通，那样运行会受到巨大的打击。国家也难以进行控制。然而我相信，历史总是会向前的，这个东西在未来必然会渐渐发展起来。这更像是一种契约，一种想要战胜挑战垄断建立平等的契约。现在的社会与企业追求利益至上，我又想起了以前的一个都市传说：用水作为燃料的汽车早就已经被发明出来了，让利益的驱动无限制的将这种革新技术扼杀与封锁，终究将其归为妄想。而数字货币之于未来，必会蓬勃发展。商业政治首脑去只接受它只是时间上的问题，或者只是在等待能够重新构建他们的获利体系罢了。
