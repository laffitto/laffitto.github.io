一个给samsung watch 安装检测血压软件的小教程

偶然看见三星还未推出的手表血压测量软件，且近日已经在韩国本土正式推出，觉得十分有趣。但是自己的手表是国行的，推送过来估计都猴年马月了，抱着折腾致死的心态，一番搜索后终于安装成功，包含手表端的tpk软件包与手机端的apk软件包。有网上能力的建议直接看原xda网站和其配套的视频教程。这里大致写一下中文教程。

注意，搞机有风险，下手请谨慎！

tpk文件安装教程（以三星血压检测软件为例）

这里手表系统调成的是英文界面，注意对照。

  1. 打开debugging：进入setting--about watch，翻到最下面，将debugging打开。

  2. 打开developer mode：进入setting--about watch--software，连续点击software version（大约7次），可以看见 developer mode turned on 的提示。

  3. 手表连接WiFi网络（注意此时要与电脑连接到同一个WiFi），关闭蓝牙连接。

  4. 解压sdb文件与tpk文件，安装tizen studio软件。

  5. 连接手表与电脑：打开Tizen Studio，在顶部会看到一个写有“No target”的框，点击下拉菜单，选择“Launch Remote Device Manager”，点击“Scan”，你会找到你的手表。点击connect进行连接。

如果无法扫描到你的手表，可以手动添加IP地址，ip地址获取方法：手表端setting--connections--WLAN--WLAN networks--自己连接的wifi名称--IP address，这里我自己的是192.168.0.102，填上点击add即可。

如果连接不成功，尝试重新启动手表连接，多手动添加几次等操作基本可以解决问题。

  1. 安装tpk软件：将tpk软件复制到sdb2260\sdb\data\tools文件目录下，点击ansicon.exe文件，输入命令：

    sdb install bp.tpk

    即可安装成功，手表上也可以看见对应的软件。

  1. 配套安装手机端的apk即可使用。

附：不用tizen studio安装的方法

网上找了找，似乎不用tizen studio也可以直接安装，并且我这边成功运行，分享一下。

上面1到4步相同，接着直接将tpk软件复制到sdb2260\sdb\data\tools文件目录下，点击ansicon.exe文件，输入命令：

sdb connect 192.168.xxx.xxx

其中192.168.xxx.xxx替换成自己手表的ip地址
如下图，显示连接成功：

再次输入

sdb install bp.tpk

即可直接安装。
